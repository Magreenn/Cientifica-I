Calculando derivadas de ordem superior¶

A primeira ideia para calcular a derivada segunda de uma funo calcular duas vezes a derivada. Inclusive porque esta a definio!

Assim, $f''(x) \sim [f'(x+h) - f'(x-h)]/2h$, e o clculo de $f'(x\pm h)$ pode ser feito analogamente.

Implemente esta frmula.

def ddf(f, x, h=1e-5):
    """Calcula uma aproximao da segunda derivada de $f$, nos pontos do vetor x, usando passos de tamanho $h$."""
    def df(f,x,H):
        return (f(x+H) - f(x-H))/(2*H)
    return (df(f,x+h,h) - df(f,x-h,h))/(2*h)

def ddf_lateral(f, x, h=1e-5):
    """Calcula uma aproximao da segunda derivada de $f$, nos pontos do vetor x, usando passos de tamanho $h$."""
    def df(f,x,H):
        return (f(x+H) - f(x))/(H)
    return (df(f,x+h,h) - df(f,x,h))/(h)
