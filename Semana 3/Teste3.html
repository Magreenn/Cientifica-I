def bissecao(f, a, b, xtol=1e-8, ytol=1e-8):
    """Algoritmo da bisseção para encontrar o zero de f.
    
    Retorna (m, niter), onde `m' é (uma aproximação para) o zero da
    função `f' no intervalo [a,b], e `n_iter' é a quantidade de vezes
    que o algoritmo calculou um ponto médio."""
    n_iter=0
    while abs(a-b)>xtol and abs(f((a+b)/2))>ytol:
        m = (a+b)/2
        if f(m)*f(a)<0:
            b = m
        else:
            a = m
        n_iter+=1
    m = (a+b)/2
    return m, n_iter

Agora, implemente o método de Newton com o seguinte critério de parada. Após cada passo, você verifica o tamanho desse passo, e o valor da função, e:

  • se o tamanho do passo que você deu foi menor do que prec, você retorna o resultado;
  • se o resultado é um ponto $p$ tal que $|f(p)|<$ ytol, você retorna esse ponto;
  • se ao dar esse passo você completou um total de maxsteps passos, você retorna o resultado.

Além de retornar o ponto $p$, a sua implementação também deve retornar a quantidade de passos que foram dados até atingirmos $p$.

def newton(f, df, x0, prec=1e-8, ytol=1e-8, maxsteps=100):
    """Método de Newton para encontrar o zero de f.
    
    Retorna (x, n_iter), onde `x' é (uma aproximação para) o zero da
    função `f', e `n_iter' é a quantidade de passos
    que o algoritmo executou.
    
    `df' é uma função que retorna a derivada de f, e `x0' é o valor
    inicial da iteração."""
    for i in range(maxsteps):
        n_iter = i+1 
        passo = f(x0)/df(x0)
        x = x0 - passo
        if abs(passo) < prec or abs(f(x)) < ytol:
            return x,n_iter
        x0 = x
    raise "O Número Máximo de Iterações foi atingido"
