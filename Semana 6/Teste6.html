# Mtodos de Cauchy, ponto mdio e simpson.
# Nomes: cauchy, midpoint, simpson
# Forma: (f,a,b,n=100)

def rect(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    return np.sum(f(l))*h

def midpoint(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    m = l+h/2
    return np.sum(f(m))*h

def simpson(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    m = l+h/2
    return (4*np.sum(f(m)) + 2*np.sum(f(l))+f(b)-f(a))*h/6
methods = [rect, midpoint, simpson]

Student's answer Score: 3.0 / 2.0 (Top)

Sabe-se que o metodo de $\frac{1}{3}$ Simpson Repetida tem erro da forma: $$\max{|f^{iv}(x)|}\frac{(b-a)h^{4}}{180}$$ e o erro do mtodo do Ponto Mdio tem a forma: $$\max{|f^{''}(x)|}\frac{(b-a)h^{2}}{24}$$.

E por estarmos aplicando esses mtodos numa funo polinomial de ordem 1 as formulas dos erros para os mtodos do $0$ j que a funo tem suas derivadas das respectivas ordens iguais a $0$ tambem.

H de se comentar o problema de no existir a derivada no ponto $0$, que faz parte do intervalo, porem este problema no influencia no mtodo j que o intervalo s definido direita desse ponto, podendo ser entendido como um "limite lateral direito".

# Faa aqui uma funo genrica para "fazer grficos de erro" para os 3 mtodos conjuntamente
# No esquea das legendas!
def graph_err(f,a,b,ans):
    ns = np.logspace(1,4,dtype=int)
    for m in methods:
        y=[m(f,a,b,ni) for ni in ns]
        plt.loglog(ns,np.abs(y-ans),label=f"{m.__name__}")
    plt.xlabel("Nmero de Passos")
    plt.ylabel("Erro")
    plt.title(f"Erro Absoluto dos Mtodos no Intervalo $[{a:.2f},{b:.2f}]$")
    plt.legend(title="Mtodo",loc="upper left", bbox_to_anchor=(1.0,1.0))
    plt.grid();

Podemos ver que o erro para o mtodo do Ponto Mdio mudou de comportamento e os outros dois se mantiveram. Um motvo para essa mudana de comportamento que, ao mudar o intervalo de integrao, o ponto onde a derivada no existe agora est no meio do intervlo, ento podemos ver que a frmula terica do erro passa a dar a forma do grfico, porem sem a derivada segunda: $$E_{PM} = \frac{(b-a)h^{2}}{24}$$

Para o mtodo de Simpson no ocorre o mesmo fenmeno pois para este mtodo utilizado uma aproximao do intervalo por uma parbola usando 3 pontos e o 0 no utilizado para isso.
