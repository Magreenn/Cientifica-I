Integrais difíceis¶

Vimos em aula que os métodos de "alta ordem" convergem mais rápido quando a função é várias vezes diferenciável. Neste teste, vamos olhar a situação contrária: vamos integrar funções que não são diferenciáveis no intervalo de integração, e ver como os métodos se comportam!

# Métodos de Cauchy, ponto médio e simpson.
# Nomes: cauchy, midpoint, simpson
# Forma: (f,a,b,n=100)

def rect(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    return np.sum(f(l))*h

def midpoint(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    m = l+h/2
    return np.sum(f(m))*h

def simpson(f,a,b,n=100):
    l,h = np.linspace(a,b,num=n,endpoint=False,retstep=True)
    m = l+h/2
    return (4*np.sum(f(m)) + 2*np.sum(f(l))+f(b)-f(a))*h/6
methods = [rect, midpoint, simpson]

Agora, explique o que aconteceu nos métodos do ponto médio e de Simpson.

Student's answer Score: 3.0 / 2.0 (Top)

Sabe-se que o metodo de $\frac{1}{3}$ Simpson Repetida tem erro da forma: $$\max{|f^{iv}(x)|}\frac{(b-a)h^{4}}{180}$$ e o erro do método do Ponto Médio tem a forma: $$\max{|f^{''}(x)|}\frac{(b-a)h^{2}}{24}$$.

E por estarmos aplicando esses métodos numa função polinomial de ordem 1 as formulas dos erros para os métodos dão $0$ já que a função tem suas derivadas das respectivas ordens iguais a $0$ tambem.

Há de se comentar o problema de não existir a derivada no ponto $0$, que faz parte do intervalo, porem este problema não influencia no método já que o intervalo só é definido à direita desse ponto, podendo ser entendido como um "limite lateral direito".

def graph_err(f,a,b,ans):
    ns = np.logspace(1,4,dtype=int)
    for m in methods:
        y=[m(f,a,b,ni) for ni in ns]
        plt.loglog(ns,np.abs(y-ans),label=f"{m.__name__}")
    plt.xlabel("Número de Passos")
    plt.ylabel("Erro")
    plt.title(f"Erro Absoluto dos Métodos no Intervalo $[{a:.2f},{b:.2f}]$")
    plt.legend(title="Método",loc="upper left", bbox_to_anchor=(1.0,1.0))
    plt.grid();

Podemos ver que o erro para o método do Ponto Médio mudou de comportamento e os outros dois se mantiveram. Um motívo para essa mudança de comportamento é que, ao mudar o intervalo de integração, o ponto onde a derivada não existe agora está no meio do interválo, então podemos ver que a fórmula teórica do erro passa a dar a forma do gráfico, porem sem a derivada segunda: $$E_{PM} = \frac{(b-a)h^{2}}{24}$$

Para o método de Simpson não ocorre o mesmo fenômeno pois para este método é utilizado uma aproximação do intervalo por uma parábola usando 3 pontos e o 0 não é utilizado para isso.

Comments:

O fato do método de Simpson não apresentar erro não tem a ver com o fato de que o "0" não é utilizado pra traçar a parábola. Vide Questão 4.

Student's answer Score: 1.0 / 3.0 (Top)

Podemos ver claramente que todos os métodos não tiveram o comportamento teórico esperado das suas fórmulas de erros máximos, já que todos tem erros de ordem $O(h)$. Este comportamento pode ser explicado pois, a função analizada é uma reta e esta sendo integrada em um intervalo simétrico, sendo assim não há ganho de precisão por calcular pontos intermediários no método de simpson, os três métodos estão fazendo exatamente a mesma operação com mais ou menos pontos.
