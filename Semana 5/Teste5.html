Integrais 2D¶

Questão 1¶

Uma integral 2D pode ser vista de 2 formas:

  • repetindo uma integral 1D ao longo de vrias "fatias", conhecida como "integral iterada";
  • uma integral numa regio 2D, dividindo a mesma em "pequenos quadrados" e somando "pequenos paraleleppedos", conhecida como "integral dupla".

Implemente o mtodo do retngulo para a integral iterada $$ \int_a^b \left( \int_c^d f(x,y) \, dy \right) \, dx $$ onde $a \leq x \leq b$ e $c \leq y \leq d$. Note que a integral "interna" pode ser vista como uma caixa-preta $g(x)$ que recebe $x$ e retorna $\int_c^d f(x,y) \, dy$.

Obs: mais fcil no esperar que $g$ assim definida seja vetorizada.

def int2_rect(f, a, b, c, d, nx=200, ny=200):
    """Integral de f no domnio  [a,b] x [c,d],  usando  nx x ny  subretngulos de integrao."""
    xis=np.linspace(a,b,num=nx,endpoint=False)
    yis=np.linspace(c,d,num=ny,endpoint=False)
    C = ((b-a)/nx)*((d-c)/ny)
    def g(x):
        return np.sum(f(x,yis))
    return np.sum(np.fromiter((g(xi)for xi in xis),float))*C

def int2_trap(f, a, b, c, d, nx=200, ny=200):
    """Integral de f no domnio  [a,b] x [c,d],  usando  nx x ny  subretngulos de integrao."""
    xis=np.linspace(a,b,num=nx,endpoint=False)
    yis=np.linspace(c,d,num=ny,endpoint=False)
    C = ((b-a)/nx)*((d-c)/ny)
    def g(x):
        return 2*np.sum(f(x,yis)) - f(x,c) + f(x,d)
    return (np.sum(np.fromiter((2*g(xi)for xi in xis),float))+g(b)-g(a))*C/4

def i2_rec(f, a, b, c, d, nx=200, ny=200):
    """Integral de f no domnio  [a,b] x [c,d],  usando  nx x ny  subretngulos de integrao."""
    axis=np.linspace(a,b,num=nx,endpoint=False)
    ayis=np.linspace(c,d,num=ny,endpoint=False)
    xs,ys=np.meshgrid(axis,ayis)
    C = ((b-a)/nx)*((d-c)/ny)
    return (np.sum(f(xs,ys)))*C
